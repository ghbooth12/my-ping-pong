## My Ping Pong

My Ping Pong is the classic game Pong replica that uses pure JavaScript and the HTML5 canvas element. It simulates a table-tennis or ping-pong match.

### How it works

User plays a game with computer. User can move the paddle with the left arrow key and right arrow key. A player earns a point when the ball gets past the opponentâ€™s paddle. The first player to earn eleven points wins.
